# Print Subsequence

## Question

 1. You are given a string str.
 2. Complete the body of printSS function - without changing signature - to calculate and print all subsequences of str.

## Input Format

- A string str

## Output Format

- Subsequences of str in order hinted by Sample output

## Sample Input
```
abc
```
## Sample Output
```

c
b
bc
a
ac
ab
abc
```
