# Power-logarithmic

## Question

1. You are given a number x.
2. You are given another number n.
3. You are required to calculate x raised to the power n. Don't change the signature of power function.

## Input Format

- A number x
- A number n

## Output Format

- x raised to the power n

## Sample Input
```
2
5
```
## Sample Output
```
32
```
