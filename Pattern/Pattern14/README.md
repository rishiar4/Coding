# Pattern 14

## Question

1. You are given a number n.
2. You've to write code to print it's multiplication table up to 10 in format given below.

## Input Format
  A number x

## Sample Input 

3

## Sample Output
```
3 * 1 = 3
3 * 2 = 6
3 * 3 = 9
3 * 4 = 12
3 * 5 = 15
3 * 6 = 18
3 * 7 = 21
3 * 8 = 24
3 * 9 = 27
3 * 10 = 30
```
